# System Hacking with Metasploit

I worked on a hands-on System Hacking project based on **CEH v13 – Module 06**. This project shows how to use the Metasploit Framework find and use exploit modules understand payloads and set up a session in a cybersecurity lab.

The project helps you understand how to use the Metasploit Framework by doing exercises in a controlled environment with Kali Linux and Metasploitable 2.

## Project Objectives

- Learn what the Metasploit Framework is used for.

- Understand how Metasploit modules are structured.

- Find and analyze exploit modules.

- Configure options for exploits.

- Learn about payloads and how they are named.

- Set up and check a Meterpreter session in a lab.

- Write a report of the assessment findings.

## Lab Environment

### Attacker Machine

- Kali Linux

### Target Machine

- metasplotiable 2

### Framework

- Metasploit Framework (msfconsole)

- Get an overview of Metasploit.

- Learn the Framework Architecture.

- Navigate the Console.

### Part 2 – Exploit Selection and Payload Understanding

- Search for Exploit Modules.

- Review Exploit Information.

- Configure Exploit Modules.

- Compatible Payloads.

- Understand Payload Naming.

### Part 3 – Exploiting the Target and Session Establishment

- Configure Target Information.

- Configure Exploit Options.

- Run the Assessment.

- Verify Meterpreter Session.

### Part 4 – Reviewing and Documenting the Assessment

- Review what happened during the Assessment.

- Document the Findings.

- Assess the Security Impact.

- Recommend Fixes.
